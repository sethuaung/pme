Key Power Query Functions

Power Query is a data preparation and ETL (Extract, Transform, Load) tool. It provides a visual, step-by-step way to clean, shape, and combine your data before it ever reaches your report. Here are some of the most common functions you'll use.

Change Type

Ensures columns have the correct data type (e.g., text, date, number) for accurate calculations and visualizations.

Remove & Choose Columns

Removes or selects specific columns to keep, simplifying your dataset and improving performance.

Filter Rows

Keeps or removes rows based on a specific condition, helping you focus on a subset of your data.

Merge Queries

Combines two queries based on a matching column, similar to a SQL join. Essential for bringing together related data.

Append Queries

Stacks two or more tables on top of each other, ideal for combining data with the same column headers from different periods or sources.

Group By

Aggregates rows into groups based on a column and performs an operation like counting or summing. A great way to get a summary view of your data.

Intro to DAX Measures

DAX (Data Analysis Expressions) is a formula language for creating custom calculations. A "Measure" is a DAX formula that calculates a result from your data, like a sum or an average. Here are a few common functions.

  • SUM: Calculates the sum of all values in a column.
    Total Revenue = SUM ( 'Sales'[Revenue] )
  • AVERAGE: Calculates the average of a column.
    Average Sales = AVERAGE ( 'Sales'[OrderQuantity] )
  • CALCULATE: The most powerful DAX function. It modifies the context in which data is filtered.
    Total Sales Last Year = CALCULATE ( [Total Revenue], PREVIOUSYEAR('Date'[Date]) )

Module 5: Key Visualization Types

Choosing the right visual is crucial for telling a clear and effective data story. Here are some of the most common and powerful chart types you'll use in Power BI.

Comparing Values

When you need to compare values across different categories, **bar and column charts** are your best choice. They make it easy to see which category is the highest or lowest.

Showing Trends Over Time

To show how a value changes over a period of time, use a **line chart**. You can easily spot trends, seasonality, and sudden spikes or drops.

Displaying Proportions

When you want to show how a total value is broken down into parts, a **donut or pie chart** is perfect. This helps visualize the composition of a whole.

Showing Relationships

A **scatter plot** is the ideal visual for exploring the relationship between two numerical variables. Each dot represents a data point, and you can spot correlations by seeing how the dots cluster.

Power BI vs. Tableau vs. Looker Studio

Each of these tools excels in different areas. Here's a quick comparison to help you understand where each one fits best in the world of business intelligence.

Power BI

Microsoft's Data Platform

  • Best For: Users heavily invested in the Microsoft ecosystem.
  • Core Strength: Powerful data modeling with **DAX** and robust data preparation via **Power Query**.
  • Ease of Use: User-friendly interface with a learning curve for advanced DAX.
  • Pricing: Freemium model with a powerful free version.
  • Key Features: Tightly integrated with Excel, Azure, and Teams.

Tableau

The Visual Storyteller

  • Best For: Interactive and beautiful data visualization.
  • Core Strength: Drag-and-drop interface for rapid, intuitive visual creation.
  • Ease of Use: Generally considered the most user-friendly for visualization design.
  • Pricing: Subscription-based, generally the most expensive.
  • Key Features: Strong geospatial analysis and excellent for storytelling.

Looker Studio

Google's Reporting Tool

  • Best For: Web-based data reporting and dashboarding.
  • Core Strength: Seamless integration with Google products like Google Analytics and Sheets.
  • Ease of Use: Very easy for beginners with a simple, web-based interface.
  • Pricing: Free to use.
  • Key Features: Great for marketing data, simple dashboards, and real-time reports.

A foundational learning path. Continuous practice is key to mastery.
