အခန်း ၁- Introduction to the Power BI Ecosystem (စနစ်သို့ မိတ်ဆက်ခြင်း)

Power BI သည် ကိရိယာတစ်ခုတည်းမျှသာမဟုတ်ပါ။ ၎င်းသည် သင့်အချက်အလက်ရင်းမြစ်များကို ခိုင်မာသော၊ အမြင်အာရုံ ထိုးထွင်းသိမြင်နိုင်သော၊ အပြန်အလှန်အကျိုးပြုသော ထိုးထွင်းဉာဏ် အဖြစ်သို့ ပြောင်းလဲရန်အတွက် အတူတကွ အလုပ်လုပ်သော ဆော့ဖ်ဝဲဝန်ဆောင်မှုများ၊ အက်ပ်များနှင့် ချိတ်ဆက်မှုများ စုစည်းမှုဖြစ်သည်။

Power BI Desktop

သင့်ရဲ့ ဖန်တီးမှု စတူဒီယို။ အစီရင်ခံစာများ ဖန်တီးရန် သင့် ဒေတာများကို ချိတ်ဆက်၊ ပြောင်းလဲပြီး မြင်သာအောင် ပြုလုပ်နိုင်သည့် အခမဲ့ Windows အက်ပလီကေးရှင်းတစ်ခု။

Power BI Service

သင့်ရဲ့ ပူးပေါင်းဆောင်ရွက်မှု ဌာနချုပ်။ အစီရင်ခံစာများနှင့် ဒက်ရှ်ဘုတ်များကို ထုတ်ပြန်ခြင်း၊ မျှဝေခြင်းနှင့် ပူးပေါင်းဆောင်ရွက်ခြင်း တို့အတွက် အသုံးပြုသော cloud-based ဝန်ဆောင်မှု (SaaS) ဖြစ်သည်။

Power BI Mobile

သင့်ရဲ့ ထိုးထွင်းဉာဏ်များကို သွားရင်း လာရင်း ရယူခြင်း။ iOS၊ Android နှင့် Windows အတွက် မူရင်းအက်ပ်များ ဖြစ်ပြီး သင့်အစီရင်ခံစာ များကို နေရာမရွေး လုံခြုံစွာ ဝင်ရောက်ကြည့်ရှုနိုင်စေရန် ခွင့်ပြုပါသည်။

Key Power Query Functions (အဓိက Power Query လုပ်ဆောင်ချက်များ)

Power Query သည် ဒေတာပြင်ဆင်မှု နှင့် ETL (ထုတ်ယူ၊ ပြောင်းလဲ၊ တင်ပို့) ကိရိယာ တစ်ခုဖြစ်သည်။ ၎င်းသည် သင့်အစီရင်ခံစာသို့ မရောက်မီ သင့်ဒေတာကို သန့်စင်ရန်၊ ပုံဖော်ရန်နှင့် ပေါင်းစပ်ရန်အတွက် အမြင်အာရုံ၊ တစ်ဆင့်ချင်းစီ နည်းလမ်းကို ပံ့ပိုးပေးပါသည်။ ဤတွင် သင်အသုံးပြုမည့် အသုံး အများဆုံး လုပ်ဆောင်ချက် အချို့ဖြစ်သည်။

အမျိုးအစား ပြောင်းပါ

ကော်လံများသည် မှန်ကန်သော ဒေတာအမျိုးအစား (ဥပမာ- စာသား၊ ရက်စွဲ၊ နံပါတ်) ရှိစေရန် သေချာစေပါသည်။

ကော်လံများကို ဖယ်ရှားခြင်းနှင့် ရွေးချယ်ခြင်း

တိကျသော ကော်လံများကို ဖယ်ရှားခြင်း သို့မဟုတ် ရွေးချယ်ခြင်းသည် သင့်ဒေတာအစုကို ရိုးရှင်းစေပြီး စွမ်းဆောင်ရည်ကို မြှင့်တင်ပေးပါသည်။

တန်းများကို စစ်ထုတ်ပါ

သတ်မှတ်ထားသော အခြေအနေ တစ်ခုအပေါ် အခြေခံ၍ တန်းများကို ထားရှိခြင်း သို့မဟုတ် ဖယ်ရှားခြင်းသည် သင့်ဒေတာ၏ အပိုင်းခွဲ တစ်ခုအပေါ် အာရုံစိုက်ရန် ကူညီပေးသည်။

မေးခွန်းများကို ပေါင်းစည်းပါ

ကိုက်ညီသော ကော်လံတစ်ခုအပေါ် အခြေခံ၍ မေးခွန်း နှစ်ခုကို ပေါင်းစည်းခြင်း၊ SQL join နှင့် ဆင်တူသည်။ ဆက်စပ် ဒေတာများကို ပေါင်းစည်းရန်အတွက် မရှိမဖြစ် လိုအပ်သည်။

မေးခွန်းများကို ထပ်ထည့်ပါ

တူညီသော ကော်လံခေါင်းစဉ်များရှိသော စားပွဲနှစ်ခု သို့မဟုတ် ထို့ထက်ပို၍ စုပုံခြင်း၊ မတူညီသော ကာလများ သို့မဟုတ် ရင်းမြစ်များမှ ဒေတာများကို ပေါင်းစည်းရန်အတွက် စံပြဖြစ်သည်။

အုပ်စုဖွဲ့ပါ

ကော်လံတစ်ခုအပေါ် အခြေခံ၍ တန်းများကို အုပ်စုဖွဲ့ပြီး ရေတွက်ခြင်း သို့မဟုတ် ပေါင်းခြင်းကဲ့သို့သော လုပ်ဆောင်ချက်ကို လုပ်ဆောင်ပါသည်။ သင့်ဒေတာ၏ အနှစ်ချုပ် ကိုရရှိရန် ကောင်းမွန်သော နည်းလမ်းဖြစ်သည်။

Intro to DAX Measures (DAX တိုင်းတာမှုများကို မိတ်ဆက်ခြင်း)

DAX (ဒေတာခွဲခြမ်းစိတ်ဖြာခြင်းဖော်မြူလာများ) သည် စိတ်ကြိုက်တွက်ချက်မှုများ ဖန်တီးရန်အတွက် ဖော်မြူလာ ဘာသာစကားတစ်ခုဖြစ်သည်။ "Measure" သည် သင့်ဒေတာမှ ပေါင်းခြင်း သို့မဟုတ် ပျမ်းမျှ ကဲ့သို့သော ရလဒ်တစ်ခုကို တွက်ချက်သည့် DAX ဖော်မြူလာတစ်ခုဖြစ်သည်။ ဤတွင် အသုံးများသော လုပ်ဆောင်ချက်အချို့ဖြစ်သည်။

  • SUM: ကော်လံတစ်ခုရှိ တန်ဖိုးအားလုံးကို ပေါင်းခြင်း။
    စုစုပေါင်းဝင်ငွေ = SUM ( 'Sales'[Revenue] )
  • AVERAGE: Cloumn (ကော်လံ) တစ်ခု၏ ပျမ်းမျှကို တွက်ချက်သည်။
    ပျမ်းမျှ ရောင်းချမှု = AVERAGE ( 'Sales'[OrderQuantity] )
  • CALCULATE: အစွမ်း အထက်ဆုံး DAX လုပ်ဆောင်ချက်။ ၎င်းသည် ဒေတာကို စစ်ထုတ်သည့် အခြေအနေကို ပြောင်းလဲသည်။
    ပြီးခဲ့သောနှစ် စုစုပေါင်းရောင်းချမှု = CALCULATE ( [Total Revenue], PREVIOUSYEAR('Date'[Date]) )

အခန်း ၅- Key Visualization Types (အဓိက အမြင်အာရုံဖော်ပြမှု အမျိုးအစားများ)

မှန်ကန်သော အမြင်အာရုံ (right visual ) ကို ရွေးချယ်ခြင်းသည် ရှင်းလင်းပြီး ထိရောက်သော (data story) ဒေတာဇာတ်လမ်းကို ပြောပြရန် အရေးကြီးပါသည်။ ဤတွင် Power BI တွင် သင်အသုံးပြုမည့် အသုံးအများဆုံးနှင့် အစွမ်းထက်ဆုံး ဇယားအမျိုးအစားအချို့ဖြစ်သည်။

Comparing Values (တန်ဖိုးများ နှိုင်းယှဉ်ခြင်း)

မတူညီသော အမျိုးအစားများအကြား တန်ဖိုးများကို နှိုင်းယှဉ်ရန် လိုအပ်သည့်အခါ၊ **ဘားနှင့် ကော်လံဇယားများ**သည် သင့်အတွက် အကောင်းဆုံးရွေးချယ်မှုဖြစ်သည်။ ၎င်းတို့သည် မည်သည့်အမျိုးအစားက အမြင့်ဆုံး သို့မဟုတ် အနိမ့်ဆုံးဖြစ်သည်ကို မြင်ရလွယ်ကူစေသည်။

Showing Trends Over Time (အချိန်ကြာလာသည်နှင့်အမျှ ခေတ်ရေစီးကြောင်းများပြသခြင်း)

တန်ဖိုးတစ်ခုသည် အချိန်ကာလတစ်ခုအတွင်း မည်သို့ပြောင်းလဲသည်ကို ပြသရန်၊ **လိုင်းဇယား**ကို အသုံးပြုပါ။ သင်သည် ခေတ်ရေစီးကြောင်းများ၊ ရာသီဥတုအလိုက် အပြောင်းအလဲများနှင့် ရုတ်တရက် တက်လာခြင်း သို့မဟုတ် ကျဆင်းခြင်းများကို အလွယ်တကူ တွေ့မြင်နိုင်သည်။

Displaying Proportions (အချိုးအစားများ ပြသခြင်း)

စုစုပေါင်းတန်ဖိုးကို အပိုင်းများအဖြစ် မည်သို့ခွဲခြမ်းသည်ကို ပြသလိုသည့်အခါ **ဒိုးနပ် သို့မဟုတ် ပိုင်ဇယား**သည် ပြီးပြည့်စုံသည်။ ၎င်းသည် တစ်ခုလုံး၏ ဖွဲ့စည်းပုံကို မြင်သာအောင် ကူညီပေးသည်။

Showing Relationships (ဆက်ဆံရေးများ ပြသခြင်း)

**ပျံ့နှံ့မှုဇယား**သည် ဂဏန်းအချက်အလက်နှစ်ခုကြား ဆက်ဆံရေးကို ရှာဖွေရန်အတွက် အကောင်းဆုံး အမြင်ဖြစ်သည်။ အစက်တစ်ခုစီသည် ဒေတာအမှတ်တစ်ခုကို ကိုယ်စားပြုပြီး အစက်များ မည်သို့ စုစည်းပုံကို ကြည့်ခြင်းဖြင့် ဆက်နွှယ်မှုကို သင်ရှာတွေ့နိုင်သည်။

6-9

အခန်း ၆-၉- Publishing & Practice Project (ထုတ်ဝေခြင်းနှင့် လက်တွေ့ပရောဂျက်)

နောက်ဆုံးအဆင့်များမှာ သင့်အစီရင်ခံစာကို Power BI Service သို့ ထုတ်ဝေခြင်းနှင့် သင်၏ ကျွမ်းကျင်မှုအသစ်များကို Power BI Service ဒေတာအစု တစ်ခုသို့ အသုံးချခြင်းဖြစ်သည်။ နမူနာ ရောင်းအားပရောဂျက် တစ်ခုကို ကြည့်ကြစို့။

Quarterly Sales Revenue Trend (သုံးလပတ် ရောင်းအားဝင်ငွေ ခေတ်ရေစီးကြောင်း)

ဤလိုင်းဇယားသည် ကျွန်ုပ်တို့၏ စုစုပေါင်းရောင်းအားဝင်ငွေသည် တစ်နှစ်အတွင်း မည်သို့ပြောင်းလဲသွားသည်ကို ပြသသည်။ ကျွန်ုပ်တို့သည် အထူးသဖြင့် final quarter နောက်ဆုံး (သုံးလပတ်) တွင် အားကောင်းသော အထက်သို့ တက် သည့် ခေတ်ရေစီးကြောင်း ( strong upward trend) ကို တွေ့မြင်နိုင်သည်။

Revenue by Product Category (ထုတ်ကုန် အမျိုးအစားအလိုက် ဝင်ငွေ)

ဤဒိုးနပ်ဇယားသည် ကျွန်ုပ်တို့၏ စုစုပေါင်း ဝင်ငွေကို ခွဲခြမ်းစိတ်ဖြာသည်။ "လုပ်ငန်းခွင်သုံးဖြေရှင်းချက်များ" သည် ကျွန်ုပ်တို့၏ ရောင်းအားအတွက် အကြီးမားဆုံး အထောက်အကူပြုကြောင်း ရှင်းလင်းစွာသိရသည်။

Power BI နှင့် Tableau နှင့် Looker Studio နှိုင်းယှဉ်ချက်

ဤကိရိယာတစ်ခုစီသည် မတူညီသောနေရာများတွင် ထူးချွန်ပါသည်။ ဤတွင် တစ်ခုစီသည် စီးပွားရေးဆိုင်ရာ ထိုးထွင်းဉာဏ်ကမ္ဘာတွင် မည်သည့်နေရာတွင် အသင့်တော်ဆုံးဖြစ်သည်ကို နားလည်စေရန် ကူညီပေးမည့် အမြန်နှိုင်းယှဉ်ချက်တစ်ခုဖြစ်သည်။

Power BI

Microsoft ၏ ဒေတာပလက်ဖောင်း

  • အကောင်းဆုံးအတွက်: Microsoft ဂေဟစနစ်တွင် အထူးအာရုံစိုက်ထားသော သုံးစွဲသူများ။
  • အဓိက အားသာချက်: **DAX** ဖြင့် အစွမ်းထက်သော ဒေတာပုံစံချခြင်းနှင့် **Power Query** မှတဆင့် ခိုင်မာသော ဒေတာပြင်ဆင်မှု။
  • အသုံးပြုရလွယ်ကူမှု: အဆင့်မြင့် DAX အတွက် သင်ယူမှုမျဉ်းရှိသော သုံးစွဲသူအတွက် အဆင်ပြေသော အင်တာဖေ့စ်။
  • စျေးနှုန်း: အစွမ်းထက်သော အခမဲ့ဗားရှင်းပါရှိသော Freemium မော်ဒယ်။
  • အဓိကအင်္ဂါရပ်များ: Excel၊ Azure နှင့် Teams တို့နှင့် တင်းကျပ်စွာ ပေါင်းစည်းထားသည်။

Tableau

အမြင်အာရုံ ဇာတ်လမ်းပြောသူ

  • အကောင်းဆုံးအတွက်: အပြန်အလှန်အကျိုးပြုသောနှင့် လှပသော ဒေတာအမြင်အာရုံဖော်ပြမှု။
  • အဓိက အားသာချက်: မြန်ဆန်သော၊ အလိုလိုသိနိုင်သောအမြင်အာရုံ ဖန်တီးမှုအတွက် ဆွဲချချပုံး အင်တာဖေ့စ်။
  • အသုံးပြုရလွယ်ကူမှု: အမြင်အာရုံဖော်ပြမှု ဒီဇိုင်းအတွက် အသုံးအဝင်ဆုံးဟု ယေဘုယျအားဖြင့် ယူဆပါသည်။
  • စျေးနှုန်း: စာရင်းသွင်းမှုအခြေခံ၊ ယေဘုယျအားဖြင့် အစျေးအကြီးဆုံး။
  • အဓိကအင်္ဂါရပ်များ: အားကောင်းသော ပထဝီဝင်ဆိုင်ရာ ခွဲခြမ်းစိတ်ဖြာမှုနှင့် ဇာတ်လမ်းပြောပြရန်အတွက် ထူးချွန်သည်။

Looker Studio

Google ၏ အစီရင်ခံကိရိယာ

  • အကောင်းဆုံးအတွက်: ဝဘ်အခြေပြု ဒေတာအစီရင်ခံခြင်းနှင့် ဒက်ရှ်ဘုတ်တည်ဆောက်ခြင်း။
  • အဓိက အားသာချက်: Google Analytics နှင့် Sheets ကဲ့သို့သော Google ထုတ်ကုန်များနှင့် ချောမွေ့စွာ ပေါင်းစည်းထားသည်။
  • အသုံးပြုရလွယ်ကူမှု: ရိုးရှင်းသော၊ ဝဘ်အခြေပြု အင်တာဖေ့စ်ကြောင့် စတင်သူများအတွက် အလွန်လွယ်ကူသည်။
  • စျေးနှုန်း: အခမဲ့ အသုံးပြုနိုင်သည်။
  • အဓိကအင်္ဂါရပ်များ: စျေးကွက်ရှာဖွေရေးဒေတာ၊ ရိုးရှင်းသော ဒက်ရှ်ဘုတ်များနှင့် အချိန်နှင့်တစ်ပြေးညီ အစီရင်ခံစာများအတွက် ကောင်းမွန်သည်။

ဤအချက်အလက်ပြဇယားသည် အခြေခံသင်ယူမှုလမ်းကြောင်းကို ပံ့ပိုးပေးပါသည်။ ဆက်လက်လေ့ကျင့်ခြင်းသည် ကျွမ်းကျင်မှုအတွက် အဓိကသော့ချက်ဖြစ်သည်။
